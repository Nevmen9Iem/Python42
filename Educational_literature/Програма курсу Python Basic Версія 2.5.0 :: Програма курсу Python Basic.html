Python Basic

Уроки

Версія 2.5.0

Тривалість курсу: 84 пари (42 дні)

Мета курсу

Навчити слухача основам програмування на Python.

Після закінчення курсу слухач буде:

  • Розбиратися в типах даних Python.
  • Розуміти тонкощі роботи зі змінними.
  • Застосовувати механізми перетворення типів даних.
  • Використовувати умови та цикли.
  • Взаємодіяти з рядками.
  • Створювати функції.
  • Розуміти особливості реалізації механізмів ООП в Python.
  • Розуміти принципи функціонального програмування.
  • Вміти використовувати систему контролю версій.
  • Застосовувати патерни проєктування.
  • Використовувати юніт-тестування.

Після закінчення цього курсу студент здає практичне завдання і складає теоретичний іспит за матеріалами курсу. Для допуску до іспиту студент має здати всі домашні та практичні завдання. Практичне завдання має охоплювати максимум матеріалу з різних розділів курсу.

Перед початком даного предмета необхідно надати студентам доступ до курсу Cisco:

  • PCAP: Programming Essentials in Python.

Цей курс вважається підготовкою до сертифікацій:

  • PCEP - Certified Entry-Level Python Programmer Certification
  • PCAP - Certified Associate in Python Programming Certification
